# week2-signals-matlab

This repository contains MATLAB code for generating and plotting basic continuous-time signals.

## Signals Implemented
- Unit Impulse Signal  
- Unit Step Signal  
- Unit Ramp Signal  
- Exponential Signal  
- Signum Signal  
- Sinc Signal
